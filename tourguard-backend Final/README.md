# TourGuard Backend (NestJS) - Generated Scaffold

This repository is a generated starting point for the TourGuard backend (Node.js + NestJS).
It includes:
- JWT auth (login/register)
- Users, Tours, Incidents modules
- TypeORM + PostgreSQL support
- Docker compose for local development
- Seed script to create admin user

## Run locally (quick)
1. Copy `.env.example` to `.env` and update values.
2. Install dependencies: `npm ci`
3. Start dev server: `npm run start:dev`
4. Or use docker-compose: `docker-compose up --build`
